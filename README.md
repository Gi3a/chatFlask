# chatFlask
Web-chat on python flask
![alt tag](https://github.com/Gi3a/chatFlask/blob/main/scren.png)

## How to install
### Creating virtual env
python -m venv venv

### Turn on venv
.\venv\Scripts\activate

### Turn off venv
deactivate

### Setup requirements
pip install -r requirements.txt

### Start app
python app\app.py

### Open browser
http://127.0.0.1:5000/
